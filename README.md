# Exploring Customer Behavior and Segmentation with Analytical SQL

## Project Description

This project focuses on exploring customer behavior and segmentation using analytical SQL techniques applied to the Online Retail dataset. The project aims to derive meaningful insights and actionable strategies by analyzing sales trends, customer purchasing behavior, and segmentation patterns.

## Getting Started

### Prerequisites
- Access to a SQL database or software (e.g., MySQL, PostgreSQL, SQLite)
- Online Retail dataset (included in the `data` directory)


## Analysis Results

- **Q1: Exploratory Analysis on Online Retail Dataset**
  - Identified months with highest and lowest revenue.
  - Investigated least sales by product in each month, particularly in January.

- **Q2: Implementation of a Monetary Model for Customer Segmentation**
  - Segmented customers based on recency, frequency, and monetary values.
  - Identified customer groups such as Champions, Loyal Customers, and Lost.

- **Q3: Analysis on Daily Purchasing Transactions Dataset**
  - Explored maximum number of consecutive days a customer made purchases.
  - Determined average time to reach a spending threshold of 250 L.E.
